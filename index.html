About

Enriching the Bangla Typing Experience

The Avro Keyboard is a blessing for Bangla typing, but its built-in autocorrect dictionary only contains 2473 words.

With complex Bangla spelling rules, this enhanced dictionary helps both fast typists and those less familiar with Bangla spelling.

Includes correct spellings for common Bangla words and English-to-Bangla translations for frequently used English words.

This work is a tribute to the creator and developers of Avro Keyboard.

How to Use

Installation Guide

1

Open Avro Settings

Right click on the Avro icon, then go to: Tools > Avro Phonetic Options > Edit/Import AutoCorrect Entries.

Also, check the 'Dictionary mode is the default in suggestion' field and don't forget to 'Enable Auto Correct'.

2

Import the Dictionary

Click on Import and select the autodict.dct file you downloaded from this website.

3

Enjoy Typing!

You're all set! Now you can enjoy a much improved Bangla typing experience with enhanced autocorrect.

Pro Tip: Restart Avro Keyboard for changes to take effect immediately.
